## 반응형 브레이크포인트 정리

| 브레이크포인트 | 변경 사항 |
|--------------|----------|
| `>= 992px` | 전체 레이아웃, navbar-status 표시, 카드 3열 |
| `<= 991px` | navbar-status 숨김, 햄버거 메뉴 |
| `<= 860px` | 카드 2열 |
| `<= 768px` | hero-title 축소, terminal max-width 100% |
| `<= 576px` | body font 13px, 카드 1열, hero/card padding 축소 |
